# 鬼の日本史

### アプリ説明
日本史で出てくる人物を画像と共に覚えていくアプリです。

### 開発の背景
初めて一人で作成したアプリです！
受験生の頃日本史を先行していたのですが、その際人の名前を大量に覚える必要があり苦労したことがありました。

そこで、人の名前と顔両方を見ながら覚えたほうが人間の脳的に覚えやすいのではと思い、作成しました！
名前、時代、顔写真を表示しています。
