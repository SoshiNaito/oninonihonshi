# oninonihonshi
This is My first app.
This app is to study japanese history.

# アプリ説明
日本史で出てくる人物を画像と共に覚えていくアプリです。
受験生時代にまとまって覚える必要がある人物は覚えにくいなと思い作りました。
顔写真があったら覚えやすかったので、名前、時代、顔写真を表示しています。
